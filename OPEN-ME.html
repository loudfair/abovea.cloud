Quick Start

  • Clone: git clone https://github.com/loudfair/abovea.cloud.git
  • Setup: cd abovea.cloud && ./setup.sh
  • Search: source venv/bin/activate && python search.py "flight logs"
  • By name: python search.py --name "Ghislaine Maxwell"
  • Emails: python search.py --email --from "epstein"
  • AI answer: python search.py --ask "Who visited the island?"
